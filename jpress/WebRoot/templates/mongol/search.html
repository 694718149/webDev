ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ

ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ

October 22, 2015 Jane Doe

ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ ᠦᠰᠬᠦ ᠡᠴᠡ ᠰᠤᠷᠤᠭᠰᠠᠨ ᠦᠨᠳᠦᠰᠦᠨ ᠬᠡᠯᠡ ᠮᠠᠷᠲᠠᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠰᠣᠶᠤᠯ ᠦᠬᠦᠲᠡᠯ᠎ᠡ ᠤᠷᠤᠰᠢᠬᠤ ᠲᠦᠷᠦᠯᠬᠢ ᠨᠤᠲᠤᠭ ᠰᠠᠯᠵᠤ ᠪᠣᠯᠤᠰᠢ ᠦᠭᠡᠢ ᠣᠷᠤᠨ
